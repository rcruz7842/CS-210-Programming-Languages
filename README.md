# CS-210-Programming-Languages#

This repository contains projects and assignments that I completed for CS-210 Programming Languages at Southern New Hampshire University.

Throughout this course, I worked with C++ and learned how to build programs using classes, functions, file input and output, exception handling, and other programming concepts. The projects in this repository show the progress I made throughout the course and how my problem-solving and coding skills developed over time.

Some of the projects included:
- Chada Tech Clocks
- Airgead Banking
- Corner Grocer Item Tracker

These projects helped me gain more experience writing, testing, and debugging C++ programs while following software development best practices.
